О EconomyBot

EconomyBot — это инновационный Discord бот, разработанный для создания engaging-экономики на вашем сервере. Бот сочетает в себе несколько взаимосвязанных систем, которые обеспечивают долгосрочную вовлеченность участников.

С помощью EconomyBot вы можете создать vibrant виртуальную экономику, где участники могут зарабатывать валюту, развивать бизнес, заниматься майнингом и многое другое.

Основные команды

💵 Экономика

  • !balance - Проверить баланс
  • !work - Выполнить работу
  • !transfer - Перевести деньги
  • !shop - Открыть магазин
  • !buy - Купить предмет

💼 Бизнес

  • !business create - Создать бизнес
  • !business hire - Нанять сотрудника
  • !business upgrade - Улучшить бизнес
  • !business profit - Получить прибыль

⛏️ Майнинг

  • !mining start - Начать майнинг
  • !mining upgrade - Улучшить оборудование
  • !mining stats - Статистика майнинга
  • !mining sell - Продать криптовалюту

🛡️ Модерация

  • !warn - Выдать предупреждение
  • !mute - Заглушить пользователя
  • !ban - Забанить пользователя
  • !clear - Очистить сообщения
